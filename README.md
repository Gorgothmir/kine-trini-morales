# 🧠 Kine Trini Morales — Landing Page

Landing page profesional para servicios de kinesiología, enfocada en captación de pacientes, presentación clara de servicios y contacto rápido vía WhatsApp.

## 🚀 Demo



---

## 🎯 Objetivo

Crear una página simple, rápida y efectiva que:

* Presente los servicios de kinesiología
* Genere confianza en potenciales pacientes
* Facilite el contacto inmediato
* Sea adaptable a móvil (mobile-first)

---

## 🛠️ Tecnologías

* React
* Vite
* CSS / Tailwind (si aplica)
* JavaScript (ES6+)

---

## 📸 Secciones de la Landing

* Hero (presentación principal)
* Sobre mí
* Servicios
* Beneficios
* Testimonios *(opcional)*
* Contacto (WhatsApp directo)
